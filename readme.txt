Admin search add-on for WP Loupe.

== Description ==

WP Loupe Admin adds an enhanced admin search experience on top of WP Loupe, including dashboard and modal search for indexed content, users, and installed plugins.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wp-loupe-admin/`, or install through the WordPress plugins screen.
2. Activate WP Loupe first.
3. Activate WP Loupe Admin through the 'Plugins' screen in WordPress.
